# 第一章 机器学习和优化

这本书的读者对机器学习或者是深度学习的概念一定不陌生，机器学习算法，尤其是深度学习，最终总是会划归成求解一个函数极小值的形式，这个函数通常我们称为损失函数，用来衡量模型预测结果是实际数据之间差异。
